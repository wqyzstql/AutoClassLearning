## 使用方法

1. 首先需要在运行设备上安装Python3运行环境

2. 接着打开命令行操作界面，CD（Change Directory）到该脚本的目录下

3. 最后执行`python main.py` 或者 `python3 main.py`即可开始运行该脚本，按照指示使用即可。
